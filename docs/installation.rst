Installation
====================================

If you just want to use a stable version of pyDelta use

.. code-block:: bash

    $ pip3 install pydelta-smt
    $ pydelta --version

If you are interested in a specific version of pyDelta (including development versions) use

.. code-block:: bash

    $ pip3 install pydelta-smt==

If you want to experiment with pyDelta, for example to test custom mutators, clone pyDelta from github and run it from your working copy:

.. code-block:: bash

    $ git clone https://github.com/nafur/pydelta.git
    $ pydelta/bin/pydelta --version
